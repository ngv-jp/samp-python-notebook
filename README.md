# samp-python-notebook
This repository is a sample for Jupyter Notebook.

## Get Started
### Execute on Linux
1. Install virtual environment
    ```bash
    python -m venv .env
    ```
1. Change virtual environment
    ```bash
    source .env/bin/activate
    ```
1. Install packages
    ```bash
    pip install -r requirements.txt
    ```
1. Execute scraping
    ```bash
    jupyter notebook
    ```
